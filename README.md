## Contenido de la aplicación

- Página de inicio con formulario para seleccionar destino, origen, fecha y cantidad de personas.  
- Página de resultados de vuelos con tarjetas de información (origen, destino, precio, horario).  
- Página final de confirmación del pasaje.  
- Sección de productos (mock API) para practicar operacion (GET).
- Estilos aplicados con SASS y estructura modular en carpetas.

## Tecnologías utilizadas

- *React 19* (con Hooks: useState, useEffect, useNavigate)
- *Vite* (para inicializar y servir el proyecto)
- *react-router-dom* (enrutamiento entre páginas)
- *SASS (.scss)* (estilos modulares)
- *localStorage* (persistencia de datos en el navegador)
- *fetch + async/await* (para comunicación con el mock)
- *json-server / mockApi.js* (simulación de API REST)
- *Alias de imports* (@components, @pages, @styles, @api, @assets)
